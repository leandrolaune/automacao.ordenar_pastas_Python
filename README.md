Projeto de uma auutomação desenvolvida em Python para ordenação de arquivos em pastas, separando tais arquivos de acordo com seus formatos.
Seguindo o exemplo abaixo:
"imagens": [".png", ".jpg", ".jpeg"]
"planilhas": [".xlsx"],
"pdfs": [".pdf"],
"csv": [".csv"],
"docs": [".docx"],
"powerpoint": [".pptx"]
